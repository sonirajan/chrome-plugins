# Chrome Plugins

Collection of personal Chrome extensions.

## Plugins

| Plugin                                                 | Description                                                                                            |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [new-yahoo-mail-remove-ad](./new-yahoo-mail-remove-ad) | Removes ads and fixes layout in new Yahoo Mail.                                                        |
| [linkedin-cleaner](./linkedin-cleaner)                 | Removes distractions from LinkedIn: member profile images, promoted posts, advertisements, and videos. |
| [serene-newtab](./serene-newtab)                       | A minimal new tab with a random full-screen background, 12-hour live clock, date, and a daily focus reminder.|



## Setup

Each plugin folder contains its own `README.md` with setup instructions.
